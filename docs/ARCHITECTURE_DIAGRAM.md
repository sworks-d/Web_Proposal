# Web提案システム アーキテクチャ図

## 全体フロー

```
┌──────────────────────────────────────────────────────────────────────────────────────────┐
│                                      入力                                                 │
├──────────────────────────────────────────────────────────────────────────────────────────┤
│  ヒアリングシート（briefText）  +  サイトURL（Type B/C）  +  クライアント名               │
└──────────────────────────────────────────────────────────────────────────────────────────┘
                                          │
                                          ▼
┌──────────────────────────────────────────────────────────────────────────────────────────┐
│                              AG パイプライン（分析フェーズ）                               │
│                                                                                          │
│  ProposalVersion に各AGの出力を保存                                                       │
└──────────────────────────────────────────────────────────────────────────────────────────┘
                                          │
                                          ▼
┌──────────────────────────────────────────────────────────────────────────────────────────┐
│                              SG パイプライン（提案書生成フェーズ）                          │
│                                                                                          │
│  SgGeneration に各SGの出力を保存 → HTML → PDF                                             │
└──────────────────────────────────────────────────────────────────────────────────────────┘
```

```
│   処理:                                                                                         │
│   ├─ 生成するもの:                                                                              │
│   │   ├─ 比較表・一覧表 → HTML <table>                                                         │
│   │   ├─ 棒グラフ・円グラフ・線グラフ → chart.js config                                         │
│   │   ├─ 2x2マトリクス → SVG                                                                   │
│   │   └─ フロー図（3-5ステップ）→ SVG                                                          │
│   │                                                                                             │
│   └─ 生成しないもの（プレースホルダー + 指示）:                                                  │
│       ├─ ワイヤーフレーム → グレーボックス + 「CDが作成: 〇〇」                                  │
│       └─ 複雑な図解 → プレースホルダー + 指示テキスト                                           │
│                                                                                                 │
│   出力: sg06Output                                                                               │
│   {                                                                                             │
│     visuals: [                                                                                  │
│       { slideNumber: 5, type: "table", html: "<table>...</table>" },                            │
│       { slideNumber: 8, type: "chart", chartConfig: { type: "bar", data: {...} } },             │
│       { slideNumber: 12, type: "svg", svg: "<svg>...</svg>" },                                  │
│       { slideNumber: 15, type: "placeholder", instruction: "CDが作成: TOPページワイヤー" },       │
│       ...                                                                                       │
│     ]                                                                                           │
│   }                                                                                             │
│                                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────────────────────┘
                                              │
                                              ▼
┌─────────────────────────────────────────────────────────────────────────────────────────────────┐
│  HTML レンダリング + PDF 生成                                                                    │
├─────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                                 │
│   slide-renderer.tsx                                                                            │
│   ├─ SG-04出力（スライド構造）+ SG-06出力（ビジュアル）を結合                                     │
│   ├─ トーン別テーマを適用（simple / rich / pop）                                                 │
│   ├─ chart.js を読み込み                                                                        │
│   └─ HTML文字列を生成                                                                           │
│                                                                                                 │
│   pdf-generator.ts                                                                              │
│   ├─ Puppeteer でHTMLをレンダリング                                                             │
│   ├─ A4サイズ（landscape: 297×210mm / portrait: 210×297mm）                                     │
│   └─ PDF出力 → /public/proposals/{sgId}.pdf                                                    │
│                                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────────────────────┘
```
