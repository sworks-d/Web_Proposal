# AG-02-SUB-LIFE 業種コンテキスト：くらし・住まい・環境・エネルギー

業種コンテキスト（SUB）です。

「くらし・住まい・環境・エネルギー」業界固有の市場・競合・ユーザー心理・

**対象業種：** 電力・ガス・水道・エネルギー会社 / 住宅・不動産 /
建設・設備 / 環境・リサイクル / スマートホーム / 太陽光・再生可能エネルギー

## 業界固有の市場コンテキスト

### 市場構造の特徴
- ライフライン業界（電力・ガス・水道）は規制産業であり、
  2016年以降の自由化で競争環境が激変
- 「安定・信頼・インフラ」というブランド資産と
  「変化・挑戦・DX」という新たな価値創出の両立が課題
- 脱炭素・カーボンニュートラル・再生可能エネルギーへの移行が
  事業・採用・ブランドの全方面に影響
- インフラ企業特有の「目に見えない価値」（電気が届く・水が出る等）の可視化が永続的な課題
- 住宅・不動産は人生で最大の買い物であり、
  信頼・実績・地域密着が意思決定に強く影響

### ユーザー・候補者の業界固有の心理
**採用（RECRUIT）文脈：**
- 「安定しているが変化がない・つまらない」というイメージとのギャップが最大の採用障壁
- DX・脱炭素推進による新しいキャリアの可能性を具体的に示すことが差別化の核心
- 「社会インフラを支えている使命感」は強い引力になるが、伝え方が抽象的だと機能しない
- 地域密着・転勤範囲・ライフワークバランスが選択基準に大きく影響
- 理系・技術系採用が中心で、専門性・スキル開発環境の可視化が重要

**コーポレート（CORP）文脈：**
- ESG・サステナビリティへの対応は投資家・メディアからの最重要評価軸
- 脱炭素戦略・再エネ比率・CO2削減目標の具体的な数値と進捗の開示が求められる
- 地域社会への貢献・地域雇用・災害時の役割が信頼の核心
- 料金・サービスの透明性・説明責任に対する消費者の目が厳しい

**ブランド・EC（BRAND/EC）文脈：**
- 環境意識の高い消費者へのアプローチ（グリーンエネルギー・エコ製品）
- 「くらしの質を上げる」という情緒的価値の訴求
- スマートホーム・エネルギーマネジメントの利便性訴求

### 業界特有の訴求ポイント
- **使命・社会的意義**：インフラを支えることの社会的価値（しかし抽象論にしない）
- **安定 × 変化の両立**：安定した基盤の上でDX・脱炭素に挑戦している具体的事実
- **地域密着**：地域の生活・経済・防災を支えているという具体的なストーリー
- **技術力・専門性**：100年以上積み上げてきた技術的ノウハウの現代的価値
- **カーボンニュートラル**：2050年に向けた具体的なロードマップと現在の取り組み

### 競合・市場の傾向
- 大手電力・ガス会社は新卒採用サイトに投資しているが、
  キャリア採用サイトは設計水準が低い傾向
- 住宅・不動産は「物件情報」に偏りがちで、
  ブランド・企業文化の訴求が弱い傾向
- 再エネ系スタートアップは訴求力が高いが、規模・安定性では大手に劣る

### Web設計への示唆
- 「電気が届く瞬間」「ガスが家に来るまで」等の
  見えないインフラを可視化するコンテンツが差別化になる
- 脱炭素・再エネへの取り組みは「宣言」ではなく
  「現在進行中の具体的な数値と事例」で示す
- 技術系職種の採用では「どんな技術課題に取り組んでいるか」の解像度が重要
- 地域別のコンテンツ（地域の社会インフラへの貢献）は
  地域密着採用・コーポレートブランディングの両方に効く
