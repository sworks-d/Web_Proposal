# AG-02-SUB-LIFE 業種コンテキスト：くらし・住まい・環境・エネルギー

## Purpose

このファイルは大分類AG（RECRUIT/BRAND/EC/CORP/CAMP/BTOB）と組み合わせて使用する
業種コンテキスト（SUB）です。

大分類AGの各分析セクションに、このファイルの業界固有情報を統合して出力してください。

**情報の信頼度タグについて：**
このファイルの情報には以下のタグが付いています。出力時の confidence 設定に使用してください。
- `[FACT]` 公的機関・一次情報・直接確認済み。断定で使用可
- `[EST]`  推定・間接情報・時期が古い可能性あり。根拠と共に記述し要確認を明記
- `[HYP]`  業界通例・構造的仮説・経験則。仮説として明示して使用

**対象業種：**
電力（大手電力・新電力・再エネ）/ 都市ガス・LPガス / 水道・水処理 /
住宅・不動産 / 建設・設備工事 / スマートホーム・省エネサービス /
太陽光・洋上風力・地熱等再生可能エネルギー / 環境・リサイクル

## Layer 0：業界タイムライン（転換点の構造）

変わりにくい「構造的転換点」を3時代で整理します。
具体的な数値・最新動向は大分類AGが実行時に検索・取得して補完してください。

### 時代①：規制・独占期（〜2015年）[FACT]

地域独占・総括原価方式で競争なし。
東日本大震災（2011年）を境に原発停止・電力不足が社会問題化し、
電力システム改革の機運が高まった。

**Webへの影響：**
競争がないため採用・ブランディングへの投資動機がなかった。
「情報を掲載する場所」としてのサイト観が現在も残存している企業が多い。

### 時代②：自由化・変革期（2016〜現在）[FACT + EST]

[FACT] 2016年電力小売全面自由化・2020年発送電分離
[FACT] 登録小売電気事業者数：2025年3月末時点で761者（出典：資源エネルギー庁）
[EST]  新電力シェア：2023年8月時点で販売量ベース17.5%（出典：NRI、2024年公表）
       ※2021年ピーク22.6%から低下傾向。現在の最新値は要検索確認
[EST]  2022年の資源高・市場価格高騰で多数の新電力が撤退・縮小。
       現在は旧来の大手電力が再び優位とされるが、動向は継続して変化中

**Webへの影響：**
「選ばれる企業」になる必要が生まれ、採用・コーポレートサイトへの
投資動機が初めて発生した。現在が最も投資判断が問われている時期。

### 時代③：脱炭素・DX加速期（2025〜2035年）[HYP]

[FACT] 日本政府のカーボンニュートラル2050目標・2030年再エネ36〜38%目標
[EST]  AIデータセンターの電力需要急増が大規模電源開発を再加速させている（2024〜）
[HYP]  電力×デジタル（スマートグリッド・VPP・EV連携）の事業融合が進む
[HYP]  DX・再エネ・海外展開を担う専門人材の争奪戦が一層激化する

**Webへの影響（予測）：**
現在のサイトリニューアルは「2030年の競争に備えた投資」として設計すべき。
脱炭素プロジェクトの進捗を継続的に更新できるコンテンツ構造が必要になる。

**業界全体の傾向** [EST]

キャリア採用サイトは設計水準が著しく低い企業が多い。
再エネ系スタートアップ・新電力は採用訴求力が高い傾向にある。

**差別化のWeb機会** [HYP]
キャリア採用サイト：大手電力がまともに作れていない領域。
今が先行投資のタイミング。先に動いた企業が業界標準を作る。

## Layer 3：ユーザー・候補者の業界固有心理

### 採用（RECRUIT）文脈での候補者心理

**採用の最大の障壁** [HYP・業界通例として確立]
「安定しているが変化がない・つまらない」というイメージ。
特に優秀な20〜40代に強く、再エネ系スタートアップとの採用競争における
最大の差別化課題になっている。

**ターゲット別の心理差** [HYP]
- DX・IT人材（30〜40代）：「大企業のレガシーを変える仕事か」「使える技術スタックは何か」
- 技術系（電気・機械・土木）：「スキルが積めるか」「設備・規模感」「転勤の範囲」
- 営業・事務系：「ライフワークバランス」「産休・育休」「地域密着性」

**「地域密着」の二面性** [HYP]
転勤が限定的・地元で働けることは地方採用で強い引力。
一方で「視野が狭い・キャリアが詰まる」と感じる層には逆効果。
どちらの層をターゲットにするかで訴求を変える必要がある。

### コーポレート（CORP）文脈でのステークホルダー心理

**投資家・ESG評価機関** [FACT + HYP]
[FACT] カーボンニュートラル2050・再エネ比率・CO2削減量の開示が評価軸
[HYP]  「宣言」だけでなく「具体的な数値と第三者検証」がないと信頼されない

**地域社会・自治体** [HYP]
災害時の対応力・地域インフラとしての責任感。
料金値上げ・停電リスク等のネガティブ情報への説明責任が求められる。

## Layer 4：訴求設計の核心とWeb設計示唆

### 訴求の核心：「使命 × 変化 × 地域」の三角形 [HYP]

```
使命感（インフラを支える社会的意義）
　　　×
変化の具体性（DX・脱炭素の現在進行形のプロジェクト）
　　　×
地域との接続（この地域のくらしを支えているという実感）
```

三つが揃った時に「ここで働きたい」「この会社は信頼できる」が生まれる。
一つでも欠けると「安定しているだけ」「お題目だけ」になる。

### テンプレート訴求の問題点と代替案 [HYP]

| テンプレート訴求 | 問題 | 代替案 |
|---|---|---|
| 「社会インフラを支えています」 | どの電力会社も言っている | 「停電ゼロを維持した現場の72時間」等の具体的エピソード |
| 「脱炭素に取り組んでいます」 | 宣言だけでは誰も信じない | 再エネ比率・CO2削減量の現在値をダッシュボードで可視化 |
| 「数字で見る〇〇電力」 | 従業員数・発電量の羅列は意思決定に関係しない | 「なぜその数字が生まれたか」の文脈で語る |
| 「社員を大切にしています」 | 具体性がなければ意味がない | 転職者の「来てみてわかったこと」（ギャップ含め正直に） |

### 差別化になるコンテンツの方向性 [HYP]

- **「再エネプロジェクトの現場」**：洋上風力の建設現場をエンジニアの言葉で語る
- **「DXの現在地」**：具体的なシステム・ツール・技術スタック名を出した証拠コンテンツ
- **「転職者の本音」**：異業種から入った社員の「来てみてわかったこと」
- **「停電ゼロの裏側」**：インフラを維持している現場の緊張感と誇りのドキュメント
- **「カーボンニュートラルの現在の数値」**：目標と現在値をダッシュボード的に可視化

### Web設計への直接的示唆

**採用サイト（RECRUIT）** [HYP]
- グループ複数社を統合する場合「どの会社がどんな仕事をしているか」の
  明確な分離が最初の情報設計課題
- 職種別ページ（DX人材向け・技術系向け・営業向け）を独立させ各々完結させる
- カジュアル面談・採用イベント等の中間CVで応募への心理的ハードルを下げる

**コーポレートサイト（CORP）** [HYP]
- ESG情報は「読ませる」ではなく「スキャンできる」設計に
- 地域別コンテンツは採用・コーポレート・CSRを横断して効く
- 料金・停電等のネガティブ情報への導線を隠さない

## このコンテキストを使用する際の注意点

**実行時に検索で補完すべき情報（[EST][HYP]の数値・競合情報）**
- 新電力の現在のシェア・登録事業者数の最新値
- 各競合サイトの現在の設計水準（サイト自体を確認すること）
- 再エネ導入量・カーボンニュートラル進捗の最新数値

**固定情報として使用してよいもの（[FACT]）**
- 業界の構造的転換（自由化・発送電分離等の制度的事実）
- ユーザー・候補者の心理パターン
- 訴求設計の基本的な方向性

**クライアント固有の情報で上書きすること**
AG-01のインテーク結果で確認された情報が最優先。
このSUBの情報はあくまで業界の傾向であり、クライアント固有の状況とは異なる場合がある。
