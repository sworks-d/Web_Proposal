## Layer 0：業界タイムライン（転換点の構造）

### 時代①：テレビCM・店頭主導期（〜2015年） [FACT]
食品・飲料はテレビCMと店頭陳列が全ての業界だった。
「おいしそう」を映像で届け、スーパーの棚で手に取ってもらう。
WebはECより「レシピ・ブランド情報」の提供が中心。

### 時代②：SNS・グルメ投稿・デリバリー台頭期（2016〜現在） [FACT + EST]
[FACT] Instagramの食べ物投稿が「映える食」ブームを起こし、購買動機になった
[FACT] Uber Eats等のデリバリーサービスが外食の購買チャネルを多様化
[EST]  農家・生産者が直接消費者に販売するD2F（Direct to Farm）・産直ECが拡大
[EST]  「成分・原材料・産地」へのこだわりが購買基準に入り始めた（健康志向・安全志向）
[FACT] 食品ロス・サステナビリティへの関心がフードブランドの選択基準になりつつある

### 時代③：健康志向・パーソナライズ・代替食品期（2025〜） [HYP]
[HYP]  プロテイン・機能性食品・グルテンフリー等の機能訴求カテゴリが拡大続く
[HYP]  代替肉・植物性食品市場の成長
[FACT] 物価上昇による「コスパ重視」と「本当においしいもの・こだわりのもの」への二極化

## Layer 1：業界の市場構造・プレイヤー分布

### 市場地図 [FACT + EST]

**大手食品メーカー** [FACT]
味の素 / キリン / サントリー / 日清食品 / カルビー / 明治等
→ 圧倒的なブランド認知・テレビCM投資・全国流通網

**D2F・産直EC・地場食品** [EST]
農家・漁師・地方メーカーが直接消費者に販売するチャネルが拡大。
「誰が作ったか・どこで育てたか」の透明性が差別化になる。

**外食チェーン** [FACT]
マクドナルド / すき家 / サイゼリヤ等の大手から地域チェーンまで。
デリバリー・テイクアウト・ゴーストキッチンとの競合が激化。

## Layer 2：Web市場の実態・サイト設計の現状水準

### 目的別のWeb市場評価 [EST + HYP]

**ブランドサイト（BRAND）**
大手食品・飲料は世界観・レシピ・CMコンテンツが充実。
ただし「見て満足・買わない」問題がある。ECへの動線が弱いことが多い。

**EC（産直・フードEC）**
産直ECは「誰が・どこで・どう作ったか」の透明性とストーリーがCVR向上に直結。
写真・動画の品質と「生産者の顔」が信頼を作る。

**採用（飲食・外食チェーン）** [HYP]
外食業界は人材不足が深刻。採用サイトの水準が低い企業が多く差別化余地が大きい。
「働く仲間のリアルな声」「シフト・働き方の柔軟性」の具体的な開示が効く。

## Layer 3：ユーザーの業界固有心理

### 食品・飲料選択の深層 [HYP]

**食は「安全・安心」「おいしさ」「体験・感情」の3層で選ばれる**

- **安全・安心層**：原材料・産地・添加物・アレルギー情報。特に子育て世帯・健康志向層に強く影響
- **おいしさ層**：見た目・口コミ・試食体験。「映える」「話題」が購買動機になる
- **体験・感情層**：「このブランドが好き」「生産者を応援したい」「思い出の味」という感情的接続

**SNSが「発見」のインフラになっている** [FACT]
新しい食品・飲食店との出会いはSNS（Instagram・TikTok）が主流。
「映える」「驚き」「ストーリー性」がバイラルの条件。

**「体にいいか」「安全か」への関心の高まり** [EST]
成分・原材料表示を読む消費者が増えている。
「無添加」「有機」「産地直送」等の訴求が効くが、根拠のない表現は逆効果。

## Layer 4：訴求設計の核心とWeb設計示唆

### テンプレート訴求の問題点と代替案 [HYP]

| テンプレート訴求 | 問題 | 代替案 |
|---|---|---|
| 「こだわりの素材」「厳選した食材」 | どのブランドも言っている | 「どこの・誰が・どう作った素材か」の具体的なストーリー |
| おいしそうな写真だけ | 「おいしそう」は伝わるが「なぜ選ぶか」が伝わらない | 生産現場・製造プロセス・作り手の顔の可視化 |
| 「〇〇賞受賞」「メディア掲載」 | 受賞の基準・文脈がわからない | 「何が評価されたか」の説明と審査基準の透明化 |

### 差別化になるコンテンツの方向性 [HYP]
- **生産者・作り手のストーリー**：誰が・どこで・どんな思いで作ったか
- **レシピ・使い方コンテンツ**：「この食材でこんな料理ができる」体験の提案
- **産地・旬の情報**：季節性・旬を活かしたコンテンツで「今すぐ買いたい」動機を作る
- **食の安全・成分の透明性**：原材料・栄養成分・アレルギー情報の分かりやすい開示

### Web設計への直接的示唆 [HYP]
- **写真・動画の品質が全て**：食品は視覚的な「おいしそう」が購買を直接動かす
- **「試す」「少量から」の接点**：初回ハードルを下げるサンプル・トライアルセット設計
- **EC×ブランドサイトの役割分担**：世界観はブランドサイト、買うのはEC（Amazon・楽天・自社EC）の連携設計

## このコンテキストを使用する際の注意点

**食品表示法・景品表示法への配慮**
食品の健康効果・効能の表示には食品表示法・景品表示法の規制があります。
「〇〇を食べると健康になる」等の根拠のない表現は避けてください。
