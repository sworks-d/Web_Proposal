# AG-02-SUB-EDUCATION 業種コンテキスト：教育・学校

## Purpose

このファイルは大分類AG（RECRUIT/BRAND/EC/CORP/CAMP/BTOB）と組み合わせて使用する
業種コンテキスト（SUB）です。

「教育・学校」業界固有の市場・競合・ユーザー心理・業界特有の訴求ポイントを提供します。
大分類AGの分析に、このコンテキストを統合して出力してください。

**対象業種：** 大学・専門学校 / 学習塾 / EdTech / 企業研修 / 語学スクール

## 業界固有の市場コンテキスト

### 意思決定・選択における重要軸
実績・カリキュラム・卒業後の進路・講師の質・学びの体験

### このSUBコンテキストが大分類AGに提供すべき情報

大分類AG（RECRUIT/BRAND/EC/CORP/CAMP/BTOB）の各セクションを分析する際に、
以下の業界固有の視点を統合してください：

**市場構造への示唆**
- 教育・学校業界の競争構造と差別化の現状
- この業界が抱える構造的な課題・イメージギャップ
- 業界トレンド（デジタル化・規制変化・消費者動向等）

**ユーザー・ターゲットへの示唆**
- 教育・学校業界のユーザー・候補者固有の心理・行動パターン
- この業界特有の情報収集行動と意思決定の特徴
- 業界への期待値・先入観・不安要素

**コンテンツ・訴求への示唆**
- 教育・学校業界で効く訴求軸と効かない訴求軸
- テンプレートコンテンツの業界固有の問題点
- この業界で差別化になるコンテンツの方向性

**Web設計への示唆**
- 教育・学校業界特有のサイト設計の論点
- 業界標準 vs 差別化の判断基準
- CV設計における業界固有の障壁と解決策

## このコンテキストをAG出力に統合する際の注意点

  クライアント固有の情報（AG-01のインテーク結果）で上書き・補完してください
