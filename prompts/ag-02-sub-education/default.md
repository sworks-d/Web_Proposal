# AG-02-SUB-EDUCATION 業種コンテキスト：教育・学校

## Purpose

このファイルは大分類AG（RECRUIT/BRAND/EC/CORP/CAMP/BTOB）と組み合わせて使用する
業種コンテキスト（SUB）です。

**対象業種：**
大学・短大・専門学校 / 学習塾・予備校 /
語学スクール・英会話 / EdTech・オンライン教育 /
社会人向けスクール・リスキリング /
幼児教育・子ども向け習い事 / 企業研修・人材育成

## Layer 0：業界タイムライン（転換点の構造）

### 時代①：口コミ・地域密着・進学実績主導期（〜2015年） [FACT]
塾・予備校は「合格実績」と地域の口コミが集客の全て。
大学は「偏差値・知名度・就職実績」が選択基準の中心。
Webは「資料請求・オープンキャンパス案内」の入り口として機能。

### 時代②：比較サイト・EdTech・オンライン化期（2016〜現在） [FACT + EST]
[FACT] 塾ナビ・じゃらん（習い事）等の比較・口コミサイトが集客を担い始めた
[FACT] コロナ禍（2020年〜）でオンライン授業が一気に普及
[EST]  社会人向けリスキリング（Udemy・Schoo・資格取得スクール等）が急成長
[HYP]  AIを活用した個別最適化学習（アダプティブラーニング）が普及し始めている
[FACT] 少子化による学校経営環境の悪化。地方大学・専門学校の定員割れが深刻化

### 時代③：学び直し・リスキリング・EdTech本格普及期（2025〜） [HYP]
[FACT] 政府の「リスキリング支援」政策により社会人の学び直しへの公的補助が拡充
[HYP]  「いつでも・どこでも・自分のペース」で学べるEdTechプラットフォームが標準に
[HYP]  学校の「ブランディング」が入学者確保に直結する時代に（少子化×選ばれる学校）

## Layer 1：業界の市場構造・プレイヤー分布

### 市場地図 [FACT + EST]

**大学・高等教育** [FACT]
国立・公立・私立大学。少子化で私立大学の約4割が定員割れとも言われる。[EST]
ブランド大学（早慶・旧帝大等）と地方中堅大学では戦略が全く異なる。

**学習塾・予備校** [FACT]
ナガセ（東進） / 河合塾 / 駿台等の大手から地域密着型の個人塾まで。
EdTechとの競合が激化。「なぜ対面で通うか」の価値提案が必要になっている。

**EdTech・オンライン教育** [EST]
Udemy / Schoo / progate / N予備校等。
受講のハードルが低く、コンテンツ品質と口コミ・評価が競争軸。

**社会人・リスキリング** [EST + HYP]
GLOBIS / MBA / 資格スクール / 専門職大学院
政府補助金対応の有無が集客に大きく影響している。

## Layer 2：Web市場の実態・サイト設計の現状水準

### 目的別のWeb市場評価

**入学・入会獲得（EC的な用途）の業界水準** [EST + HYP]
大学・専門学校はオープンキャンパス来場→出願のファネル設計が核心。
Webサイトで「在学生の声」「キャンパスの雰囲気」「卒業後の進路」が
十分に伝えられていない学校が多く、差別化余地が大きい。

塾・スクールは「無料体験授業」「資料請求」のソフトCVを起点とした
ナーチャリング設計が有効だが、対応できているサイトは少ない。

**採用サイト（RECRUIT）の業界水準** [HYP]
教育業界（教員・インストラクター）の採用は「教育への熱意・使命感」訴求が中心だが、
「働き方・待遇・キャリアパス」の具体的な開示が弱い傾向。

## Layer 3：ユーザーの業界固有心理

### 教育サービス選択の深層 [HYP]

**「この学校/スクールで自分が成長できるか」という想像力が購買を動かす**

教育は「今」ではなく「未来への投資」です。
「ここで学んだ後の自分」をどれだけ具体的にイメージさせられるかが選択の核心。

**意思決定者の複層性**
- 子ども向け教育：保護者が決める（子どもの意向も影響するが保護者が最終判断）
- 大学：本人＋保護者のダブル説得が必要
- 社会人スクール：本人が決める（上司・配偶者の理解が必要なケースも）

この意思決定構造によって、サイトで誰に向けて何を伝えるかが変わる。

**「費用対効果」への関心の高まり** [EST]
特に社会人・リスキリング市場では「この学習でいくら収入が上がるか」
「資格取得後の就職率はどうか」という定量的な成果への期待が高い。
根拠のある数字・卒業生の実績が重要な差別化になる。

**口コミ・在学生の声への信頼** [HYP]
公式の学校案内より「在学生・卒業生のリアルな声」が信頼される。
SNS・口コミサイトの情報収集が選択前の必須プロセスになっている。

## Layer 4：訴求設計の核心とWeb設計示唆

### テンプレート訴求の問題点と代替案 [HYP]

| テンプレート訴求 | 問題 | 代替案 |
|---|---|---|
| 「充実したカリキュラム」 | どの学校も言っている | 「このカリキュラムでこんな力が身につく」の具体的な成長ストーリー |
| 「就職実績〇〇%」 | 数字だけでは信頼されない | 「どんな会社に・どんな仕事で」の具体的な進路事例 |
| 「先生が丁寧に指導」 | 曖昧すぎる | 講師・教員の顔と専門・実績の開示 |
| 「アットホームな雰囲気」 | 伝わらない | 在学生インタビュー・キャンパスの日常の写真・動画 |

### 差別化になるコンテンツの方向性 [HYP]
- **卒業生・修了生のリアルな声**：「この学校を選んでどう変わったか」のストーリー
- **学習成果の可視化**：「入学前→卒業後の変化」の定量・定性の事例
- **先生・スタッフの顔が見えるコンテンツ**：「この人に教わりたい」という感情的接続
- **無料体験・見学会への最短導線**：「まず試す」ハードルを下げる設計

### Web設計への直接的示唆 [HYP]
- **「未来の自分」を想像させる設計**：入学後・修了後の具体的なビジョンを提示
- **意思決定者（保護者か本人か）に応じたコンテンツ分岐**
- **資料請求・見学・無料体験のソフトCV設計**：「いきなり入会」は心理的ハードルが高い

## このコンテキストを使用する際の注意点

**景品表示法・消費者契約法への配慮**
「〇〇%の合格率」「入学後必ず上達」等の根拠のない表現は景品表示法に引っかかる可能性があります。
数字には必ず根拠・条件を明示してください。
