## Layer 0：業界タイムライン

### 時代①：スペック・機能主導期（〜2015年） [FACT]
「性能が高い＝売れる」時代。カタログ・スペック表が購買の判断材料。
WebはカタログのデジタルHTML版として機能。比較サイトが台頭し始めた。

### 時代②：体験・エコシステム・スマホ主導期（2016〜現在） [FACT + EST]
[FACT] Apple・Googleのエコシステム戦略が「スペックより体験・連携」の購買基準を作った
[FACT] スマートフォンへの機能集約でデジカメ・カーナビ等のカテゴリが縮小
[EST]  EV・スマートホーム・IoTで「家電と自動車とデジタルの境界」が溶け始めている
[HYP]  AIアシスタント内蔵・生成AI機能搭載が次の差別化軸に

### 時代③：AIデバイス・サステナビリティ期（2025〜） [HYP]
[HYP]  AI PC・AI スマホ等の「AI搭載」カテゴリが本格化
[FACT] 右肩下がりのPC市場でAI需要による買い替え需要が発生しつつある
[HYP]  修理しやすさ・長期サポート・リサイクル設計等のサステナビリティが選択基準に

## Layer 1：市場構造・プレイヤー分布 [FACT + EST]

**グローバル大手**
Apple / Samsung / Sony / Panasonic / シャープ / 富士通等
→ グローバルブランド vs 日本市場特化の二軸。海外勢との競争が激しい領域

**通信キャリア**
NTTドコモ / au（KDDI） / SoftBank / 楽天モバイル
→ 端末販売からサービス・コンテンツへのシフトが加速。料金・回線品質に加えてアプリ・サービスの充実が差別化軸

**B2B IT機器**
富士通 / NEC / 日立 / Lenovo等
→ 法人向けはRFP・入札が主流。WebはBtoBの初期情報収集・信頼確認ツールとして機能

## Layer 2：Web市場の実態 [EST + HYP]

**ブランドサイトの水準**
大手はスペック・世界観の両方に投資しているが、「買う理由」への言語化が弱い傾向。
「このデバイスで自分の生活がどう変わるか」の体験訴求が弱いことが多い。

**EC・比較サイトとの関係**
価格.com・Amazonでのスペック比較が購買の主流。
公式サイトは「ブランドの信頼確認」と「最新情報の取得」に使われる傾向。

---

## Layer 3：ユーザーの深層心理 [HYP]

**テックデバイス購買の動機は「能力の拡張」と「ライフスタイルの表現」**
- スペック追求層：スペック・ベンチマークで合理的判断。価格比較サイトを熟読
- ライフスタイル層：「このデバイスを使っている自分」のイメージが購買動機
- ブランドロイヤル層：Apple信者・Sony信者等。ブランドへの帰属意識

**「難しくない」の証明がCVの鍵** [HYP]
新しい技術・機能への「使いこなせるか不安」という心理的障壁がある。
「これは難しくない・すぐ使える」の実証がCV前の必須ステップ。

---

## Layer 4：訴求設計の核心とWeb設計示唆 [HYP]

**テンプレート訴求の問題と代替**
| テンプレート | 問題 | 代替案 |
|---|---|---|
| スペック表の羅列 | 一般消費者には判断できない | 「このスペックで何ができるか」の体験を動画で |
| 「最新・最高性能」 | 競合も言っている | 「この機能で日常がこう変わる」の具体シーン |

**Web設計への示唆**
- 動画デモ・使用シーンの映像が購買決定に直結する
- 既存製品からの「乗り換え理由」を明確にする（買い替え需要の喚起）
- B2B用途なら「導入事例・ROI・サポート体制」への導線が最重要
